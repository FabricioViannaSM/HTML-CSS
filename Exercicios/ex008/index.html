Principais formatações

Negrito/destaque

Nesta frase temos um termo em negrito usando a tag <b> e </b>(não semântica).

Nesta frase temos um termo em negrito usando a tag <strong> e e</strong> (semântica).

Itálico/Ênfase

Nesta frase temos o termo em itálico usando as tags <i> e </i> (não semântica).

Nesta frase temos o termo em itálico usando as tags <em> e </em> (semântico).

Texto marcado

Nesta frase temos um termo marcado usando as tags <mark> e </mark>.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.(apenas big não é semântico)

Texto deletado

Podemos marcar um texto excluído usando as tags <del> e </del> para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto inserido usando as tags <ins> e </ins> para mostrar que ele foi adicionado depois.

Existe também o texto sublinhado usando as tags <u> e </u>.(não semântico)

texto sobrescrito

Usamos as tags <sup> e </sup> para deixar um texto sobrescrito. Exempo: x20 + 3

texto subscrito

Usamos as tags <sub> e </sub> para deixar um texto subscrito. Exemplo: H2O
